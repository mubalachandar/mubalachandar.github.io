About Me

I am an aspiring Cloud Security Engineer focused on securing cloud infrastructure in Amazon Web Services (AWS). I am building practical experience in cloud security, identity and access management, infrastructure protection, logging, and monitoring through hands-on labs and projects.

My work includes managing IAM users, groups, roles, and policies, securing EC2 compute instances, protecting S3 storage, configuring VPC network security, and monitoring cloud activity using AWS CloudTrail and CloudWatch.

I focus on implementing least-privilege access, secure cloud architecture, and continuous monitoring to protect cloud resources from misconfigurations and unauthorized access.
